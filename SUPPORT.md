# Support

Use the smallest public channel that fits the situation.

- Questions, setup help, and troubleshooting: start a Q&A discussion.
- Early ideas and workflow sketches: start an Ideas discussion.
- Board screenshots, reusable templates, and reading workflows: use Show and tell.
- Reproducible bugs: open a bug report issue with steps, version, platform, and redacted sample data if needed.
- Docs, translation, or README drift: open a Docs or localization issue.
- Security vulnerabilities: do not open a public issue with details. Follow `SECURITY.md`.

Live demo: https://guesswhat-studio.github.io/Calabash/

GitHub Discussions: https://github.com/Guesswhat-Studio/Calabash/discussions

Issue chooser: https://github.com/Guesswhat-Studio/Calabash/issues/new/choose

Please remove private reading notes and clearly mark story spoilers before posting screenshots, exports, or templates.
